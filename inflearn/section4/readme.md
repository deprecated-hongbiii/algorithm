## 3. 멘토링

(2021.07.02)

- 어떻게 풀어야 할 지 고민하다가 문득 그래프로 풀어봐야겠다는 생각이 들었다.
- 연결관계를 표현하기 위해 2차원 배열을 사용했다.
- 연결되어 있으면 1이라고 표시해주고, 서로 (양방향) 연결되어 있으면 다시 2로 바꿨다.
- 그럼 한 방향으로 연결된 애들만 남는데 그 인덱스를 출력하면 끝~
